### 🛡️ Scientific Summary
**YOU ARE HERE** is a mechanical stabilization framework designed to bypass compromised executive function during acute cognitive friction. It is not therapy or self-help; it is a **diagnostic and somatic manual** for returning the human hardware to a functional baseline.

The core axiom: **If intent does not equal output, the hardware is misaligned—the operator is not malformed.** 

Research suggests that when cognitive friction exceeds a specific threshold (measured on the Sanding Scale), internal narratives become unfactual. This toolkit provides the somatic overrides needed to stabilize the system until high-fidelity judgment can be restored.
